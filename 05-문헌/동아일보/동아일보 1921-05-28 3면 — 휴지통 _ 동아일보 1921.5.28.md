```
휴지통
▲소위상업학교의학도가 선생문뎨로 동맹휴학하얏다는사정은 따로히 보도한바로 그행동에대하야 별로칭찬할일도업지마는 일이이에 이르기까지에는 그내면에대하야 학도의 리약이를드르면 다소사정도잇다▲선린상업학교와소의상업학교가다 가치남대문밧게잇는까닭으로 아츰저녁에 두학교의학도가뎐차안에모히는일이만흔바 소년의일이라 서로맛나면각기학교자랑을하는것이 전례가되얏다▲그런데 선린상업학교는 재정이넉々힌 까닭으로범백의설비가 조흔고로 소의상업학교생도는 항상그아래에 눌니우기쉬웁다▲그리고 그우에선린상업학교 출신이 교사중에한사람이 잇는고로"나도졸업만하면 너의선생이된다"하는롱담에는 매우불쾌히 생각하야 오든터인데 이번에 그교사가사임하자 다시 선린학교출신이오는까닭에반대가이러난모양이다▲그럼으로 이번 동맹휴학은교사그사람을 배척하는이보다는 두학교학도가 평시에자긔학교를자랑하는말속에서불평이생긴모양이다 소년의일은 항용이□히기도쉬웁다할는지
```
