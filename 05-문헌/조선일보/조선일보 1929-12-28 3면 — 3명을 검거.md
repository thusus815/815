## 본문 — 원문 (한자)

```
三名을檢擧
충남공주읍내(忠南公州)삼중등 학교가접종하야 동맹휴학을 단행한이래 경찰의경계가 자못삼엄하다함은 루차보도한바와 갓거니와더욱 당지영명학교(永明學校)의맹휴에 대하야는 배후에혹무엇이나잇는가하야 주목을 게을니아니하든 경찰은돌연히 지난십사일 오후부터 충청남도경찰부와협력하야 대활동을 개시하고영명학교 학생층 일학년한승주(韓聖柱)삼학년 김진철(金鎭徹)동두명과 공주 소년동맹(公州少年同盟)집행위원장 안병두(安秉斗)군을검속하고 경관대가 지휘하야 전긔세명의가택을 위시하야 모—든사회단체의간판이 함끠붓허잇는 공주청년회관(公州靑年會舘)등을 전후세시간이나 엄밀한 수색을하야몃가지 문서를압수하야 간후전긔검속자에 대하야는 자못엄중한 취조를거듭하는일방 중남경찰부에서는 소택 고등과장을위시하야 동차석등이 공주경찰서까지 출동하야 철야토록말의하는등 시내공긔가 매우긴장한중이며 동검속자는 이튼달까지석방치아니하고 그대로 류치취조충이더라(공주)
```
