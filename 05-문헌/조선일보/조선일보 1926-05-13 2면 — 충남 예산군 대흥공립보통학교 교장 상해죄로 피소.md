# 조선일보 1926-05-13 2면 — 충남 예산군 대흥공립보통학교 교장 상해죄로 피소

## 본문 — 원문 (한자)

```
大興普校長
傷害罪로被訴
생도가동맹휴학한결과
선동하얏다고무리구타
충남례산군대흥면동서리백칠십이번디(忠南禮山郡大興面東西里一七二)리학구(李鶴求)(二四)는대흥공립보통학교장산본무이(大興公立普通學校長山本武二)(三一)를 상대자로 지난팔일에상해고소(傷害告訴)를공주디방법원홍성지텽검사국(公州地方法院洪城支顧檢事局)에 뎨출하엿다는데 그자세한내용을 듯건대지난 삼월경에 동교생도가모훈도의 배척으로 동맹휴학을한일이잇섯는데 전긔 산본교장은지난 사일오후열시경에 대흥읍내김규순(金圭淳)의집에서전긔피해자 리학구를맛나서 무단히구타하며 말하되 네가삼월경에생도를 권고하야 동맹휴학을하게하엿다하며 무수구타한바 전긔리학구(李鶴求)는 절대그와가튼사실이 업다하야 일대활극을연출 하엿고 겸하야산본교장은 그동생산본행헌(山本幸憲)과그의 처남 풍영중행(妻男豊永重幸)을 시키여 권총(拳銃)을 가저오라하며 이놈을쏘아죽이겟다고하며무수구타하야 류혈이림리하게되야 한참은위험하게되엿든바대흥주재소순사가 현장에와서 중재하엿슴으로성명을보존하야가지고례산을내공의에게와서일주일치료들요한다는진단서를 마터서이 를첨부하야검사국에고소를 뎨출 한것이라더라(례산)
```
