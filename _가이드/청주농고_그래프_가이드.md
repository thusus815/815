# 청주농고 항일운동 그래프 별도 보기

## Obsidian 그래프에서 청주농고만 보는 방법

### 방법 1: 로컬 그래프 (추천)
1. `청주농업고등학교_항일운동_허브` 노트 열기
2. 오른쪽 상단 **점 3개 메뉴** → **로컬 그래프 열기**
3. 깊이(Depth)를 **2~3**으로 설정
4. → 허브 노트에서 연결된 391명 항일인물이 방사형으로 표시됨

### 방법 2: 전체 그래프 필터
1. 그래프 뷰 열기
2. **필터** → 검색창에 `tag:청주농고` 입력
3. → 청주농고 관련 노트만 초록색으로 표시

### 방법 3: Dataview로 표 보기
`청주농업고등학교_항일운동_허브` 노트 내 Dataview 쿼리가 자동으로 관련 인물 표를 생성합니다.
