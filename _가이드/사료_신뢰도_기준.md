# 사료 신뢰도 기준

이 볼트에 입력되는 모든 정보는 **검증 가능한 사료**에 근거해야 합니다.

## 사료 유형 코드 (source_type)

| 코드 | 유형 | 예시 | 검증 방법 |
|------|------|------|-----------|
| `공문서` | 국가기관 공식 문서 | 조선총독부 판결문, 수형기록 | 국가기록원 문서번호 확인 |
| `국가기록원` | 국가기록원 소장 자료 | 판결원본 PDF | archives.go.kr URL |
| `공훈전자사료관` | 독립유공자 공훈록 | 공훈록 상세페이지 | e-gonghun.mpva.go.kr URL + mngNo |
| `법원판결문` | 재판 판결 원문 | 대전지방법원 청주지청 판결 | 국가기록원 CJA 문서번호 |
| `정부보고서` | 정부 연구·조사 보고서 | 친일반민족행위진상규명 보고서 | 발행기관 + 발행연도 |
| `학술논문` | 학술지 게재 논문 | KCI 등재 논문 | DOI 또는 RISS URL |
| `단행본` | 출판된 단행본 | 충북 학생 항일 독립투쟁사 | ISBN |
| `신문기사` | 당대 또는 현대 언론 | 동아일보, 조선일보 | 기사 URL 또는 날짜+매체명 |
| `외교사료관` | 일본 외무성 외교사료관 | 소요사건 개황 | 자료명 + 소장처 명시 |
| `미확인` | 출처 불명확 | — | 입력 불가 (검증 필요로 분류) |

## 신뢰도 등급 (source_verified / source_reliability)

| 등급 | 기호 | 기준 | 처리 |
|------|------|------|------|
| 1등급 | `●●●` | URL 또는 문서번호로 원문 직접 접근 가능 | 정상 통과 |
| 2등급 | `●●○` | 발행기관·날짜·제목 명확, 원문 접근 가능성 높음 | 정상 통과 |
| 3등급 | `●○○` | 출처 있으나 원문 미확인 (재인용, 2차 자료) | 정상 통과 (주의 표시) |
| 미확인 | `○○○` | 출처 불명 — inbox 제출 시 자동 격리 | `_inbox/검증필요/` 이동 |
| 레거시 | `레거시` | 검증 파이프라인 도입 이전 생성 파일 | 사료 출처 재확인 권장 |

## 제출자 (submitted_by)

모든 파일은 `submitted_by` 필드로 제출자를 기록합니다.

| 값 유형 | 예시 | 설명 |
|---------|------|------|
| 사람 이름 | `홍길동` | 직접 입력한 연구자 |
| Agent ID | `claude-agent-001` | LLM 에이전트 자동 생성 |
| 시스템 | `시스템_이관` | 검증 파이프라인 도입 전 파일 일괄 이관 |
| 미기재 | `미기재` | 제출자 불명 (검증 필요) |
