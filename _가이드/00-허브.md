## 4. LLM 추천 워크플로우

| 목적 | 시작 노트 | 다음 단계 |
|---|---|---|
| **인물 조사** | 해당 인물 노트 → "관련 인물" 섹션 따라가기 | 그래프 깊이 2~3 |
| **사건 분석** | `02-사건/<사건>` → 관련 인물·기관·지역 | `path:인물 OR path:사건` 필터 |
| **시기 분석** | [[마스터_인덱스]] → 연도 클릭 | `tag:연도/1919` 필터 |
| **지역 분석** | `04-지역/<지역>` → 로컬 그래프 | 깊이 3 |
| **새 인물 추가** | `AGENTS.md` § 6-1 (Ingest) 절차 | 출처 frontmatter 필수 |
| **위키 점검** | `AGENTS.md` § 6-3 (Lint) 절차 | 죽은 링크·중복·고아 정리 |

## 5. 그래프 뷰 빠른 명령

| 인물-기관 네트워크 | `path:"01-인물" OR path:"03-단체"` |
| 사건-인물 네트워크 | `path:"02-사건" OR path:"01-인물"` |
| 학교-인물 (충북) | `path:"07-항일학교" OR (path:"01-인물" tag:충북)` |
| 시기 1919 | `tag:연도/1919` |
| 메타 제외 (기본) | `-path:"00-원자료" -path:"_가이드" -path:"99-attachments"` |

## 6. 작업 원칙 요약 (긴 버전: AGENTS.md)

1. **L1 원자료는 절대 수정/이동하지 않는다.**
2. **모든 위키 노트에 `source` frontmatter 필수.**
3. **죽은 링크는 본문에서 빼고 `## 새로 발견된 연결` 섹션으로 이동.**
4. **연도·분류 태그만 쓰고 그래프에는 노출하지 않는다.**
5. **그래프 노이즈는 `.obsidian/graph.json`으로만 조정한다 (수동 노트 삭제 ❌).**
