# 그래프 뷰 전환 가이드

> [!info] 그래프 뷰를 두 가지 모드로 전환하는 방법

---

## 모드 전환 방법

Obsidian 그래프 패널 왼쪽 상단의 **필터(Filters)** 와 **색상 그룹(Color Groups)** 을 활용합니다.

---

## 👤 모드 1 — 인물 중심 뷰

> 친일인물·항일인물 관계망만 보기. 문서·사전 노드 숨김.

**그래프 패널 → Filters → Search 창에 입력:**
```
path:인물/ OR path:항일인물/
```

또는 불필요한 폴더 제외:
```
-path:스냅샷/ -path:기관/ -path:친일잔재/
```

**색상 의미:**
| 색상 | 의미 |
|------|------|
| 🔴 빨강 | 친일인물 (991명) |
| 🔵 파랑 | 항일인물 (451명) |
| 🟡 노랑 | 기관 |
| 🔴 진빨강 | 사건 |

---

## 📚 모드 2 — 문서/사전 중심 뷰

> 책·조사서·잔재물 문서 노드 중심으로 보기. 인물 노드 축소.

**그래프 패널 → Filters → Search 창에 입력:**
```
path:친일잔재/ OR path:항일학교/ OR path:스냅샷/
```

**색상 의미:**
| 색상 | 의미 |
|------|------|
| 🟠 주황 | 친일잔재 문서 |
| 🟣 보라 | 항일학교 |
| 🟢 초록 | 스냅샷(연도별) |

---

## 🗺️ 모드 3 — 지역·사건 레이어 뷰

> 지역과 사건을 허브로, 인물이 어디에 연결되는지 보기.

**그래프 패널 → Filters → Search 창에 입력:**
```
path:지역/ OR path:사건/ OR path:인물/
```

---

## 🔍 로컬 그래프 (개인 노드 중심)

특정 인물 노드를 열고 → 우측 상단 **"Open local graph"** 클릭하면
그 인물과 직접 연결된 노드만 보입니다.

깊이(Depth)를 **1~3** 으로 조절하면 1단계 연결 → 3단계 연결까지 확장됩니다.

---

## ⚙️ 빠른 필터 설정법

1. 그래프 뷰 열기 (좌측 사이드바 그래프 아이콘)
2. 좌측 패널 **Filters** 펼치기
3. **Search** 창에 위 쿼리 붙여넣기
4. 즉시 적용됨 (저장 불필요)

### 자주 쓰는 필터 쿼리 모음
```
# 인물만
path:인물/ OR path:항일인물/

# 충북 출신 친일인물만
tag:충북출신

# 충북 조사서 수록 인물
tag:충북조사서

# 연도 1919년 활동 인물
tag:연도/1919

# 특정 인물 제외
-path:스냅샷/ -path:기관/

# 잔재물 관련만
tag:충북잔재물

# 새로 발견된 연결
tag:새연결
```
