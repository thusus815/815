# 그래프 뷰 컨트롤 패널

> [!tip] 두 가지 방법으로 원하는 뷰 선택
> **방법 A** — 아래 링크를 클릭하고 우측 상단 **"로컬 그래프 열기"** → 해당 허브 중심의 네트워크
> **방법 B** — 그래프 뷰 검색창에 아래 쿼리 붙여넣기

---

## 👤 인물 탭

| 선택 | 클릭 (로컬 그래프) | 그래프 검색 쿼리 |
|------|-------------------|----------------|
| 친일인물 전체 | *(그래프 검색창 사용)* | `path:인물/` |
| 항일인물 전체 | *(그래프 검색창 사용)* | `path:항일인물/` |
| 친일 vs 항일 대비 | *(그래프 검색창 사용)* | `path:인물/ OR path:항일인물/` |
| 충북 출신 친일 | *(그래프 검색창 사용)* | `tag:#충북출신` |
| 충북 조사서 수록 | *(그래프 검색창 사용)* | `tag:#충북조사서` |

## 📖 책별 뷰 탭

> 특정 책(불망·파묘·충북학생항일)에서 발굴된 내용만 보기

### 불망 (김남균 저) — 충북 지역 친일잔재 현장 취재

| 보기 방식 | 방법 |
|-----------|------|
| 불망 챕터 노드만 | 그래프 검색창에 아래 쿼리 입력 |
| 불망 + 연결 인물 | 챕터 노트 클릭 → 로컬 그래프 Depth 2 |

그래프 검색 쿼리:
```
file:불망
```

연결 인물까지 함께:
```
file:불망 OR (path:인물/ OR path:항일인물/)
```

### 파묘 (김남균 저) — 민영휘 관련 친일 재산 추적

| 보기 방식 | 방법 |
|-----------|------|
| 파묘 노드만 | 아래 쿼리 |
| 파묘 + 민영휘 연결망 | 파묘 노트 클릭 → 로컬 그래프 Depth 2 |

그래프 검색 쿼리:
```
file:파묘
```

파묘 + 연결 인물:
```
file:파묘 OR (path:인물/ OR path:항일인물/)
```

### 불망 + 파묘 동시에 (두 책 모두)

```
file:불망 OR file:파묘
```

두 책 + 연결 인물 전체:
```
file:불망 OR file:파묘 OR path:인물/ OR path:항일인물/
```

## ⚙️ 자주 쓰는 쿼리 모음

```
# 기본 (가이드·스냅샷 제외)
-path:_가이드/ -path:스냅샷/

# 인물만
path:인물/ OR path:항일인물/

# 충북 출신 친일
path:인물/ tag:#충북출신

# 지역 허브 중심
path:지역/ OR path:인물/ OR path:항일인물/

# 사건 허브 중심
path:사건/ OR path:인물/ OR path:항일인물/

# 기관 허브 중심
path:기관/ OR path:인물/

# 항일 네트워크만
path:항일인물/ OR path:항일학교/ OR path:사건/

# 잔재물 관련
tag:#충북잔재물 OR path:친일잔재/

# ── 책별 뷰 ──────────────────────────
# 불망 챕터만
file:불망

# 파묘 챕터만
file:파묘

# 불망 + 파묘 두 책 모두
file:불망 OR file:파묘

# 불망 + 연결 인물
file:불망 OR path:인물/ OR path:항일인물/

# 파묘 + 연결 인물
file:파묘 OR path:인물/ OR path:항일인물/

# 불망·파묘 + 연결 인물 전체
file:불망 OR file:파묘 OR path:인물/ OR path:항일인물/
```
