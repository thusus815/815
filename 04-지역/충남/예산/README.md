# 충남 예산 지역 항일운동 자료

## 주요 사건

### 예산공립농업학교 동맹휴학 (1924, 1926)
- 1924년: 5년제 승격 요구 동맹휴학
- 1926년 6월: 교육 차별 규탄 동맹휴학 → 2학년 전원 퇴학

### 예산공립농업학교 적색독서회·좌익협의회 사건 (1932~1933)
- 강봉주를 중심으로 독서회 → 좌익협의회 → 예산학생동맹으로 발전
- 1932년 11월 만경좌 공연 저지 후 대거 피검
- 1933년 공주지방법원·경성복심법원 판결
