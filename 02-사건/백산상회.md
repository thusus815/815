# 백산상회 (白山商會, 1914~1928)

> **안희제(安熙濟)가 부산에 설립한 상회(商社). 겉으로는 무역·곡물 회사이나 실제로는 대한민국 임시정부에 군자금을 전달하는 비밀 조달망이었다. 14년간 임정의 재정을 뒷받침하다 일제의 감시와 자금 소진으로 1928년 청산되었다.**

## 개요

| 항목 | 내용 |
|------|------|
| 설립 | 1914년, 부산 |
| 설립자 | [[항일/안희제\|안희제]] (安熙濟, 1885~1943) |
| 확대 | 1919년 백산무역주식회사로 개편 (자본금 100만 원) |
| 역할 | 임정 군자금 조달·전달, 독립운동가 연락망 |
| 주주 | 경남·부산 지역 민족자본가 다수 참여 |
| 청산 | 1928년 — 일제 감시 강화·재정 소진 |

## 안희제와 대동청년당의 연결

백산상회의 설립자 안희제는 [[사건/대동청년당|대동청년당]] (1909) 창립 멤버이며,
[[사건/신민회|신민회]] 계열 인물이다.

- **1909** — 대동청년당 창립 (김태희·남형우·안희제 등 80여 명)
- **1914** — 부산에서 백산상회 설립 (자본금 3,000원)
- **1919** — 3·1운동 이후 백산무역주식회사로 개편, 임정과 공식 연계
- **1928** — 청산

## 군자금 전달 방식

```
부산 백산상회
    │
    ├── 국내 영업망 (서울·원산·대구·봉천 지점)
    │       └── 상거래 이익금을 군자금으로 위장 전달
    │
    └── 임정 연락망 (상하이)
            └── 안희제 본인이 직접 상하이 방문, 전달
```

일제는 1920년대 내내 백산상회를 감시했으나 정식 기업 형태를 띠었기 때문에
1928년까지 완전히 차단하지 못했다.

## 안희제의 말년

백산상회 청산 후 안희제는 만주로 이동, 발해 연구와 독립운동을 이어갔다.
1943년 일제에 의해 체포되어 **함흥형무소에서 옥사**했다.

사후 **건국훈장 독립장** 추서 (1962).

## 의의

백산상회는 한국 독립운동사에서 **"기업을 통한 독립운동"**의 대표 사례다.
무장·외교 노선과 달리, 합법적 상업 활동으로 임정을 재정 지원한 제3의 방식이었다.

대동청년당 → 신민회 계열의 인맥이 임정 지원망을 조직적으로 유지한 사례로,
본 볼트의 충북 항일 흐름(신민회 → 대동청년당 → 임정/의열단/백산상회)과 직접 연결된다.
