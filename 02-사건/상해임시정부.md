# 대한민국 임시정부 (1919~1945)

> **1919년 3·1운동 직후 중국 상하이에서 수립된 한국 최초의 민주공화제 임시정부. 1945년 광복까지 26년간 독립운동의 최고 구심체 역할을 했다. 본 볼트에서는 특히 충북 출신 인물들의 임정 참여와 연통제 활동에 초점을 맞춘다.**

## 개요

| 항목 | 내용 |
|------|------|
| 수립 | 1919년 4월 11일, 중국 상하이 |
| 정체 | 민주공화제 (헌법 제1조: 대한민국은 민주공화제로 함) |
| 주석 | 이승만 → 이동녕 → 김구 (역대) |
| 소재 | 상하이 → 항저우 → 전장 → 창사 → 광저우 → 류저우 → 치장 → **충칭** (1940~1945) |
| 광복 귀국 | 1945년 11월 23일 (개인 자격) |

## 연통제 — 국내와의 비밀 연결망

임정은 국내 독립운동을 지원하기 위해 **연통제(聯通制)**를 운영했다.

```
임시정부 (상하이)
    │
    ├── 각 도 독판부 (총독부 도와 대응)
    │       └── 충청북도 독판부
    │               └── 참사: [[항일/김태희|김태희]] (1920.09.10 임명)
    │
    └── 군자금 모집망
            └── 백산상회 (안희제) → 임정 군자금 조달
```

| 인물 | [[항일/김태희\|김태희]] (1877~1936) |
| 임명 | 1920년 9월 10일 |
| 직책 | 충청북도 독판부 참사 |
| 활동 | 군자금 모집, 독립운동 문서 배포, 국내 비밀 연락망 유지 |
| 출신 | 청주 청남학교 설립자, 신민회·대동청년당 계열 |

## 신민회 → 대동청년당 → 임정 흐름

```
1907 신민회 창립
    │
1909 대동청년당 (외곽 청년조직)
    │
1919 ┌─ 3·1운동
     │
     └─ 대한민국 임시정부 수립 (4월)
             │
             ├── 남형우     → 법무총장
             ├── 안희제     → 군자금 조달망
             ├── 김태희     → 충북 독판부 참사
             └── 곽재기     → 의열단 (무장노선)
```
