## 3. 그래프 뷰 추천 명령

옵시디언 좌측 메뉴 → **그래프 뷰** 열고, 검색창에 다음을 넣어보세요:

| 보고 싶은 것 | 검색식 |
|---|---|
| 인물 전체 (친일+항일) | `path:"01-인물"` |
| 친일만 (항일 제외) | `path:"01-인물" -path:"01-인물/항일"` |
| 항일만 | `path:"01-인물/항일"` |
| 인물 ↔ 단체 | `path:"01-인물" OR path:"03-단체"` |
| 1919년 관련 | `tag:"연도/1919"` |
| 충북 관련 (지역+학교+인물) | `path:"04-지역/충북" OR path:"07-항일학교" OR tag:충북` |
| 메타 노트 숨기기 | `-path:"_가이드" -path:"00-원자료" -path:"99-attachments"` |

## 5. 인물 노트 구조 (참고)

각 인물 노트는 다음 형태:

```
---
이름: 김영철
한자: 金永哲
생몰: 1841 ~ 1923
관련인물: [이완용]
관련기관: [매일신보, 조선귀족령, 조선총독부]
side: 친일
출처: 친일반민족행위진상규명_보고서
---

<!-- LLM-WIKI:SUMMARY:START -->     ← LLM이 본문 읽고 만든 요약
## 한 줄 요약
## 핵심 행적
## 같은 흐름의 인물
<!-- LLM-WIKI:SUMMARY:END -->

# 김영철 (金永哲, 1841~1923)
... 보고서 원문 그대로 (200~500줄) ...
```

**원문은 절대 손대지 않습니다.** 요약 블록만 자동 삽입.
